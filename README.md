# sbugs

## Requirements
### pycparser version 2.20

```sh
$ pip3 install pycparser==2.20
```

## Usage
#### Test Generator
```sh
$ ./testGenerator.py -o <outfile> <inputfile>
$ ./testGenerator.py -h #Help
```

#### Pre-Processor
```sh
$ ./PreProcessor.py -o <outfile> <inputfile>
$ ./testGenerator.py -h #Help
```
